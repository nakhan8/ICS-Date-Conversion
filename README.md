# ics-date-converter

Calendar integrations with SnapHabit. "ICS" is a calendar format. 

It converts the old ics file into new ics file with an update date. 

### how to run program
1. Clone repo: git clone 

2. then: cd destination-folder

3. npm install

4. npm start 

5. localhost:8008

6. Select ics file

7. Select new date

8. Download new file
